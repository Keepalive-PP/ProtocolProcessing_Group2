Detailed Description

Router module.

The module combines DataPlane, ControlPlane, RoutingTable, and Interface modules to a single Router module. In addition, the router module offer a set of methods to extract data from the submodules. The router builds the submodules and binds their ports. The number of interfaces in the router is defined in the configuration object that is passed to the router as constructor argument.
